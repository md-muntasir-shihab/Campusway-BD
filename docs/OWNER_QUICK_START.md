# Owner Quick Start (Non-Technical)

Date: 2026-03-04

## 1) Daily Admin Access
- Open: `/__cw_admin__/login`
- Sign in with admin account.
- Go to Dashboard for quick health checks.

## 2) What To Check First Each Day
- Reports: `/__cw_admin__/reports`
- Support unread count: `/__cw_admin__/support-center`
- Pending payments: `/__cw_admin__/payments`
- News pending review: `/__cw_admin__/news`

## 3) Core Content Updates
- Site logo/name/footer: `Settings -> Site Settings`
- Home section visibility + featured items: `Settings -> Home Control`
- Social links: `Settings -> Site Settings` (Social Links block)
- Subscription plans: `Subscription Plans`

## 4) Student Experience Controls
- Exam gating/profile rules: `Settings -> Home Control` and exam settings.
- Notification automation: `Settings -> Notifications`
- Analytics privacy/toggles: `Settings -> Analytics`

## 5) Safe Publishing Flow
1. Save draft/update.
2. Preview in public pages.
3. Confirm mobile view.
4. Publish/enable.

## 6) Quick Recovery
- If a page looks stale, do one hard refresh.
- If still stale, check:
  - browser console errors
  - admin save response
  - report widgets
- For persistent issues, use `docs/TROUBLESHOOTING.md`.
