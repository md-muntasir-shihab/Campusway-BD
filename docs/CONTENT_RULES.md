# Content Rules

Date: 2026-03-04

## Branding Assets
- Logo:
  - Preferred: transparent PNG or SVG
  - Recommended min height: 64px
  - Keep clear space around logo
- Favicon:
  - PNG/ICO square format
  - 32x32 or 64x64

## Home/Section Content
- Hero title: 35-70 characters
- Hero subtitle: 80-180 characters
- CTA text: 2-4 words

## News Content
- Title: 45-110 characters
- Summary/excerpt: 120-260 characters
- Banner image:
  - Recommended aspect: 16:9
  - Recommended size: 1280x720 or above
- Keep source attribution and publish date.

## University Content
- University name: 3-80 characters
- Short form: 2-15 characters
- Address: concise, valid location string
- Application dates: ISO date and timezone-aware
- Missing optional values should be saved as empty and rendered as `N/A`.

## Resource Content
- File name: clear and versioned (e.g., `physics-model-test-v2.pdf`)
- Keep total size practical for mobile download.
- Provide short description for discoverability.

## Subscription Plan Content
- Plan title: 3-40 characters
- Price label: use BDT format (`৳`)
- Keep benefit bullets short and scannable (max ~90 chars each).

## Language Guidance (BN/EN/Mixed)
- Keep one style per block for readability.
- Avoid mixed transliteration in headings.
- For bilingual text, use separate line per language where possible.
