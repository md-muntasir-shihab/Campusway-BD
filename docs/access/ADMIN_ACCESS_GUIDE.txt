CampusWay Admin Access Guide
Updated: 2026-03-04

1) Canonical Admin UI Route
- Admin login: /__cw_admin__/login
- Admin dashboard: /__cw_admin__/dashboard
- All admin modules: /__cw_admin__/*

2) Legacy Compatibility Routes
- /admin/login -> redirects to /__cw_admin__/login
- /admin/* -> redirects to /__cw_admin__/*
- /campusway-secure-admin -> redirects to /__cw_admin__/dashboard

3) Canonical Admin Menu Mapping (Serial, Route-Driven)
- Dashboard -> /__cw_admin__/dashboard
- Home Control -> /__cw_admin__/settings/home-control
- Site Settings -> /__cw_admin__/settings/site-settings
- Banner Manager -> /__cw_admin__/settings/banner-manager
- Universities -> /__cw_admin__/universities
- News -> /__cw_admin__/news
- Exams -> /__cw_admin__/exams
- Question Bank -> /__cw_admin__/question-bank
- Students -> /__cw_admin__/students
- Student Groups -> /__cw_admin__/student-groups
- Subscription Plans -> /__cw_admin__/subscription-plans
- Payments -> /__cw_admin__/payments
- Resources -> /__cw_admin__/resources
- Support Center -> /__cw_admin__/support-center
- Reports -> /__cw_admin__/reports
- Security Center -> /__cw_admin__/settings/security-center
- System Logs -> /__cw_admin__/settings/system-logs
- Admin Profile -> /__cw_admin__/settings/admin-profile

4) Admin API Path (Backend Secret Path)
- API base remains secret-path based:
  /api/{ADMIN_SECRET_PATH}
- Default local value is often:
  /api/campusway-secure-admin
- UI path and API secret path are intentionally separate contracts.

5) Default Credentials
- Deterministic default users are seeded via:
  backend/src/scripts/seed-default-users.ts
- For local bootstrap credentials, check:
  backend/INITIAL_ACCESS_INFO.txt
- Do not expose default credentials on public routes.

6) Security Notes
- Change admin password after first login.
- Restrict and rotate ADMIN_SECRET_PATH in production.
- Disable default setup accounts in production environment.
