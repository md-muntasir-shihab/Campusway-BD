# CampusWay - local setup and run instructions

আপনার ওয়েবসাইটটি লোকাললি রান করার জন্য নিচের ধাপগুলো অনুসরণ করুন। আমি আপনার সুবিধার্থে একটি সহজ গাইড এবং একটি অটোমেটিক স্টার্ট স্ক্রিপ্ট তৈরি করে দিচ্ছি।

## প্রয়োজনীয়তা (Prerequisites)
১. **Node.js**: আপনার পিসিতে Node.js ইনস্টল করা থাকতে হবে।
২. **MongoDB**: একটি লোকাল MongoDB ডাটাবেস বা MongoDB Atlas ক্লাস্টার থাকতে হবে।

---

## ১. ব্যাকএন্ড রান করার নিয়ম (Backend Setup)
১. প্রথমে একটি টার্মিনাল/কমান্ড প্রম্পট খুলুন।
২. ব্যাকএন্ড ফোল্ডারে যান: `cd backend`
৩. ডিপেন্ডেন্সি ইনস্টল করুন (যদি আগে না করা থাকে): `npm install`
৪. প্রোজেক্টের রুট বা `backend` ফোল্ডারে একটি `.env` ফাইল তৈরি করুন এবং নিচের তথ্যগুলো দিন:
   ```env
   PORT=5000
   MONGO_URI=mongodb://localhost:27017/campusway
   JWT_SECRET=your_super_secret_key
   NODE_ENV=development
   ```
৫. রান করুন: `npm run dev`

---

## ২. ফ্রন্টএন্ড রান করার নিয়ম (Frontend Setup)
১. আলাদা একটি টার্মিনাল/কমান্ড প্রম্পট খুলুন।
২. ফ্রন্টএন্ড ফোল্ডারে যান: `cd frontend`
৩. ডিপেন্ডেন্সি ইনস্টল করুন: `npm install`
৪. রান করুন: `npm run dev`

---

## ৩. এক ক্লিকে রান করার উপায় (Run with One Click)
আমি নিচের ফোল্ডারে `start_campusway.bat` নামে একটি ফাইল তৈরি করে দিচ্ছি। এই ফাইলটিতে ডাবল ক্লিক করলেই ব্যাকএন্ড এবং ফ্রন্টএন্ড একসাথে চালু হয়ে যাবে।

**ফাইলটির অবস্থান:** `f:\CampusWay\start_campusway.bat`
